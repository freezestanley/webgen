## 表单页方案模板

1. 页面类型 / 目标
   - 登录 / 注册 / 提交表单页，目标是低阻力完成输入与提交。
2. 目标用户 / 使用场景
   - 新用户或回访用户，常见于移动端单手操作与桌面快速录入。
3. 主要板块与信息层级
   - 标题与说明
   - 表单主体
   - 辅助说明 / 帮助入口
   - 次级操作：忘记密码、切换登录方式、返回
4. 设计拨盘
   - `DESIGN_VARIANCE = 3-5`
   - `MOTION_INTENSITY = 1-3`
   - `VISUAL_DENSITY = 4-5`
5. 配色 / 字体 / 图标
   - 保证对比度和可读性优先
   - 图标只辅助输入语义，不承担核心说明
6. 素材 / 文案 / 接口
   - 错误文案、校验规则、接口返回码需要尽量明确
7. 技术实现
   - 先保证原生可提交与校验反馈
   - 视觉动效只做轻反馈，不做重表演
8. 状态设计
   - 字段级错误
   - 提交中
   - 提交失败
   - 提交成功 / 下一步引导

## 表单页规则

- 优先保证可读性、可输入性、状态反馈清晰。
- Label 放在输入框上方。
- Error 文案放在输入项下方。
- 不允许为了视觉效果牺牲表单可用性。
- 默认不使用过重 3D 特效。
- 高级感主要来自：排版、留白、层次、边界处理、状态反馈。
