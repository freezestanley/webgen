# 火遮眼宣传官网设计文档

## 项目目标

为 2026 版电影《火遮眼》制作一个可运行、可预览的一页式中文宣传官网首版。页面只使用已知影片资料，不编造额外剧情，用海报式视觉、动作片节奏与地下犯罪气氛完成宣发表达。

## 设计结论

- 选型：A. 电影海报式主宣发页
- 融合：局部吸收地下档案感与角色对决编排
- 页面关键词：怒火、铁锤、追索、街区、血色反光、实战美学
- Design Read：电影宣传官网 / 面向动作片与类型片观众 / 暗红黑金、海报式压迫感、东南亚地下犯罪街区气质 / 偏电影海报与特刊混合编排
- DESIGN_VARIANCE：8
- MOTION_INTENSITY：5
- VISUAL_DENSITY：5

## 信息架构

1. Hero
   - 片名《火遮眼》
   - 宣发标语“怒火中的复仇”
   - 上映日期双行信息
   - 主 CTA：立即关注上映 / 查看主创阵容
2. 剧情简介
   - 王伟身份、女儿被掳、腐败警方失能、铁锤复仇
   - 记者纳文寻找失踪妻子，与王伟结盟
3. 主创阵容
   - 导演、监制、主要演员与角色名
4. 动作流派 / 角色对决
   - 围绕“五大流派巅峰对决”做视觉化卡组
   - 不写未提供的具体招式剧情，只强调力量对位与动作卖点
5. 上映信息
   - 中国内地、中国香港、北美上映日期
6. 奖项 / 分级
   - TIFF Midnight Madness 观众票选奖第三名
   - 香港 III 级、北美 R 级、内地引进版
7. Visual CTA
   - 以红色光带、口号与日期收束页面情绪

## 视觉策略

- 主色：off-black、深血红、旧金色
- 辅助：烟灰、暗铜、微弱象牙白
- 构图：大标题压场 + 纵深渐变 + 抽象海报块 + 细网格纹理
- 避免：紫蓝科技光效、均匀三栏卡片、廉价玻璃风、外链版权海报

## 交互与动效

- 轻量入场：Hero 标题、信息条、区块分层渐显
- 按钮反馈：active 压感、描边亮起
- 背景动效：微弱脉冲光晕与纹理漂移
- 降级：尊重 `prefers-reduced-motion`，禁用持续动画

## 响应式策略

- PC：左右分栏 + 大面积留黑 + 海报式排版
- Pad：保留层级，减少装饰性偏移
- H5：严格单栏，关键信息前置，不让标题与日期被背景吃掉
- 所有触屏设备：核心信息常显，不依赖 hover

## 内容边界

- 不添加未给出的详细情节、角色背景或评论口碑
- 不使用外部真实剧照或海报
- 不接远端 API

## 验证目标

- 页面可本地构建
- 页面可本地预览并返回 200
- 页面首屏与主要区块在产物中可检索
- PC / Pad / H5 版式逻辑在代码层已覆盖
